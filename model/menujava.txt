[MainFrame.java]
JFrame 상속받은 전체 프레임 구성
구성요소는 왼쪽: 카테고리 버튼, 중앙(menuPanel): 메뉴 아이템 영역, 
오른쪽(optionPanel): 옵션 표시 영역, 하단(selectedLabel, cartPanel): 선택된 메뉴 표시, 장바구니 영역

COFFEE/NON-COFFEE 버튼 누르면 해당 카테고리 메뉴 표시
MenuItemPanel 사용해 메뉴 표시
사용자가 메뉴 클릭하면 옵션 패널 나타남
selectedLabel- 현재 선택 상태 보여줌, CartPanel-장바구니 항목 보여줌
StartDialog- 안내 표시

주요 메서드
showMenus- menuPanel, optionPanel 초기화+ 선택된 카테고리의
메뉴 불러와서 menuItemPaenl로 메뉴 영역에 추가, 각 메뉴는
클릭 시 showOptions(item)호출되도록
showOptions- 해당 메뉴의 옵션 설정용 JDialog 생성(4가지 옵션 입력)
각 옵션 버튼은 JRadioButton으로 구현되어 있음+누르면 배경색 바뀌는 연출 포함

[MenuBoard.java]
전체 메뉴 항목을 담아두고, 각 카테고리에 맞는 메뉴만 주는 기능

전체 메뉴 항목을 담고 있는 리스트 items
static final-고정 리스트, 실행 시 한 번만 생성
MenuItem- 이름, 가격, 카테고리, 이미지 정보를 가진 객체

MenuItem 객체들을 리스트에 등록함
static 블록은 MenuBoard 클래스가 처음 메모리에 로드 될때 한 번만 실행됨
(미리 정의된 항목들은 프로그램 실행되면 메뉴로 표시될 준비 완료)

Catregory 열거형(COFFEE, NON_COFFEE) 받아서 stream API 사용해서 리스트 필터링
filter(~)-해당 카테고리만 남기고 나머지는 제거
collect(~)-List<MenuItem>형태로 변환해서 리턴

[MenuItem.java]
메뉴 하나의 정보를 담는 객체
각 메뉴가 가져야 할 기본 정보 4가지 저장(이름, 가격, 카테고리 구분, 이미지 경로)
item.getPrice() 이런식으로 정보를 얻을 수 있음


[MenuItemPanel.java]
MenuItem의 정보를 UI 패널로 그려줌
메뉴 항목 하나를 보여주는 작은 컴포넌트(클릭할 수 있고, 이미지/이름/가격 표시)
그래서 두 가지 인자를 받는다
MenuItem item- 메뉴 정보를 담고 잇는 객체(이름, 가격, 이미지파일명)
Runnable onClick- 패널이 클릭되었을 때 실행할 작업 담은 함수

마우스를 올리면 손 모양 커서를 보여주는 효과
폴더에서 이미지 파일 가져올때 이미지를 리사이징, imageLabel로 만들어 가운데 정렬
item.getName()으로 이름 가져옴
addMouseListener- 클릭 이벤트 등록, 클릭하면 onClick.run()호출됨(옵션 창 띄우는 람다 실행됨)

[MenuStyle.java]
JPanel 상속, 기본 Swing 패널을 확장해서 커스텀 스타일을 입힌 패널
둥근 모서리, 배경색, 테두리, 공통 폰트 설정

paintComponent(Graphics g)메서드는 Swing 컴포넌트가 화면에 그려질 때 자동 호출됨
RoundRextangle2D 객체- 모서리를 둥근 사각형으로, fill()로 내부 색상, draw()로 외곽선 그림
anitialiasing 설정- 경계 부드럽게 보이도록 함
getCafeCont()- 폰트 제공

[StartDialog.java]
Dialog 창을 이용해 메뉴판 보여주기 전에 "메뉴를 선택하세요" 안내 메시지 팝업 창 보여주는 역할

JDialog 상속해서 작은 팝업 창 역할
사용자가 확인을 누르면 닫히는 구조(중앙에 메시지, 아래쪽에 확인 버튼)
인자로 messageColor 받은 색상 적용
확인 버튼은 JButton으로, 클릭시 dispose()- 다이얼로그 종료됨

JPanel btnPanel- 버튼을 담는 하단 패널
